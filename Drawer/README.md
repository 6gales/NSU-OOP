# Drawer (Task #5)
## Inheritance, pure virtual functions, variadic arguments using templates and drawing using bitmap
[Original task](https://avoronkov.gitlab.io/oop/2018.cpp/task5/)
## Build
```
$ cd build
$ cmake ..
$ make
```

## Launch
```
$ ./drawer
$ ./drawer in.txt
```
## Test

```
$ ./unittest
```
